# Сырые скриншоты для гайдов (жителя и исполнителя)

Клади сюда **реальные скриншоты бота** с именами из списка ниже. После этого я
обрежу, обрисую (стрелки, подсветка нужной кнопки, номера шагов, подписи) и вставлю
их в гайды вместо нарисованных мокапов.

## Как снимать
- Формат **PNG** (можно JPG). Имя — строго из списка ниже (иначе переименую).
- В кадр должно попасть **сообщение бота + кнопки** (можно весь экран телефона —
  обрежу сам). Интерфейс — **русский**.
- Замажь/избегай личных данных (реальные ФИО, телефоны, точные адреса) — или не
  переживай, я закрою на этапе обрисовки.
- Если какого-то экрана нет или он выглядит иначе — просто пропусти, подскажу дальше.

## Житель (`res-*`)
| Файл | Что на экране |
|---|---|
| `res-menu.png` | Главное меню жителя (видны кнопки «Создать заявку», «Мои заявки») |
| `res-category.png` | Экран выбора категории заявки |
| `res-address.png` | Экран выбора адреса (двор / дом / квартира) |
| `res-confirm.png` | Сводка заявки перед отправкой (кнопка «Подтвердить») |
| `res-created.png` | Сообщение, что заявка создана, с её номером |
| `res-status.png` | «Мои заявки» или карточка со статусом *(опционально)* |
| `res-accept.png` | Уведомление «Выполнена» + кнопки «Принять»/«Вернуть» и оценка |

## Исполнитель (`exe-*`)
| Файл | Что на экране |
|---|---|
| `exe-menu.png` | Меню исполнителя (кнопка «Принять смену») |
| `exe-pool.png` | Пул свободных заявок / список заявок |
| `exe-card.png` | Карточка заявки «В работе» с кнопками (Выполнить / Материалы / Вернуть) |
| `exe-materials.png` | Выбор материала для списания |
| `exe-qty.png` | Ввод количества материала *(опционально)* |
| `exe-done.png` | Экран завершения (отчёт + фото) или сообщение «Выполнена» |

## Приоритет (если снять всё некогда)
Обязательный минимум — эти 8: `res-menu`, `res-category`, `res-confirm`,
`res-accept`, `exe-menu`, `exe-card`, `exe-materials`, `exe-done` (они заменят
текущие 8 мокапов один-в-один). Остальные — бонус.

> Как закинешь — просто напиши «скрины на месте», и я всё обрисую и вставлю.
