# Скриншоты на узбекском (для узбекского варианта гайда жителя)

Клади сюда **скриншоты того же потока, что и в русском гайде, но с интерфейсом на узбекском**.
Имена можно любые (хоть дефолтные «Снимок экрана …») — я сам распознаю каждый экран,
обрежу, замаскирую личные данные, обрисую и вставлю в узбекскую версию инструкции.

## Как переключить на узбекский
- **Бот:** Профиль → «Редактировать профиль» → 🌐 язык → **O'zbek**. Или в мини-приложении:
  вкладка **Профиль → Til → O'zbek**.
- После переключения все кнопки станут узбекскими — снимай те же экраны.

## Что нужно (18 экранов — те же, что в русском гайде)

### Бот (чат), 5 кадров
1. **Bosh menyu** — главное меню (кнопка «Ariza yaratish» / создать заявку)
2. **Kategoriya** — экран выбора категории заявки
3. **Manzil** — экран выбора адреса
4. **Tasdiqlash** — сводка заявки перед отправкой (кнопка подтвердить)
5. **Mening arizalarim** — список «Мои заявки»

### Мини-приложение (TWA), 13 кадров
6. **Bosh sahifa** — главная (часы работы / контакты)
7. **Yaratish → kategoriya** — создание, шаг «категория»
8. **Yaratish → manzil** — шаг «адрес»
9. **Yaratish → tavsif** — шаг «описание проблемы»
10. **Yaratish → foto** — шаг «добавить фото»
11. **Yaratish → shoshilinchlik** — шаг «срочность»
12. **Yaratish → tasdiqlash** — шаг «подтверждение»
13. **Arizalar** — список заявок (Faol / Arxiv)
14. **Ariza kartochkasi** — карточка одной заявки
15. **Qabul qilish** — вкладка «Приёмка»
16. **Ruxsat → avto** — контроль доступа, вкладка «Авто»
17. **Yangi ruxsatnoma** — оформление пропуска (такси/гость/доставка)
18. **Profil** — вкладка «Профиль» (язык, адреса)

## Правила
- Формат PNG (можно JPG). Русский текст в кадре не нужен — только узбекский интерфейс.
- Личные данные (имя, телефон) можно не прятать — замажу при обработке.
- Если какого-то экрана нет или он выглядит иначе — просто пропусти.
