# Глоссарий

> _Последнее редактирование: 2026-07-06_

Короткие объяснения ролей, статусов заявок и терминов, которые встречаются в
системе.

## Роли

| Роль | Кто это |
|---|---|
| **Житель** (заявитель) | Жилец дома, подаёт заявки в УК через бота |
| **Исполнитель** | Мастер, уборщик и т.п. — выполняет назначенные заявки |
| **Обходчик** | Сотрудник, осматривает территорию и заводит заявки «в поле» |
| **Менеджер** | Диспетчер УК — принимает, распределяет и контролирует заявки |
| **Оператор охраны** | Сотрудник КПП — контролирует въезд/выезд на территорию |
| **Системный администратор** | Менеджер с расширенными правами: настройка оборудования контроля доступа, пользователей и ролей |
| **Администратор** | Общий управленческий администратор (по правам близок к менеджеру) |

> Один человек может иметь несколько ролей и переключаться между ними.

## Статусы заявки

Заявка проходит по этим статусам (их видно и жителю, и менеджеру):

| Статус | Что означает |
|---|---|
| **Новая** | Заявка создана, ждёт назначения исполнителя |
| **В работе** | Назначен исполнитель, работа идёт |
| **Уточнение** | Исполнителю/менеджеру нужны детали от заявителя |
| **Закуп** | Для выполнения нужны материалы, идёт закупка |
| **Выполнена** | Исполнитель закончил работу и отчитался |
| **Исполнено** | Работа подтверждена и закрыта |
| **Принято** | Заявитель принял результат (при необходимости с оценкой) |
| **Возвращена** | Результат не устроил — заявка возвращена в работу |
| **Отменена** | Заявка отменена |

## Термины

| Термин | Значение |
|---|---|
| **Номер заявки** | Человекочитаемый номер вида `ГГММДД-NNN` (например `260706-001`) |
| **Канбан-доска** | Экран менеджера, где заявки разложены по колонкам-статусам и их можно перетаскивать |
| **Назначение** | Прикрепление исполнителя к заявке — вручную менеджером или авто-подбором |
| **Пул заявок** | Общие заявки, которые исполнитель может взять сам |
| **Групповое назначение** | Одна заявка на нескольких исполнителей |
| **Приёмка** | Подтверждение заявителем (или менеджером за него), что работа выполнена |
| **Возврат** | Отправка заявки обратно в работу, если результат не устроил |
| **Смена** | Рабочий интервал исполнителя; заявки распределяются с учётом смен |
| **Контроль доступа** | Модуль въезда/выезда: распознавание номеров машин, пропуска, шлагбаумы |
| **Склад материалов** | Учёт закупок и списания материалов на заявки с расчётом остатков |
| **Дашборд** | Веб-панель управления (для менеджеров, админов, охраны) |
| **Табло жителей** | Публичная страница УК с объявлениями, без входа |
