# Руководство пользователя UK Management

> _Последнее редактирование: 2026-07-06_

Пользовательская документация системы управления заявками жилого комплекса.
Здесь собраны инструкции для всех, кто работает с системой — без технических
подробностей. Выберите свою роль.

## С чего начать

- **[01. Начало работы](01-getting-started.md)** — что это за система, где
  работать (Telegram-бот или веб-дашборд), как войти, как сменить язык.

## Инструкции по ролям

| Кто вы | Где работаете | Инструкция |
|---|---|---|
| **Житель** — подаёте заявки в свою УК | Telegram-бот | [02. Житель](02-resident.md) · [O'zbekcha](02-resident.uz.md) |
| **Исполнитель** — мастер/уборщик, выполняете заявки | Telegram-бот | [03. Исполнитель](03-executor.md) |
| **Менеджер** — диспетчер УК, контролируете работу | Веб-дашборд + бот | [04. Менеджер](04-manager.md) |
| **Обходчик** — осматриваете территорию, заводите заявки | Telegram-бот | [05. Обходчик](05-inspector.md) |
| **Охрана (КПП)** — контролируете въезд/выезд | Веб-панель охраны | [06. Оператор охраны](06-security-operator.md) |
| **Системный администратор** — настраиваете доступ и пользователей | Веб-дашборд | [07. Системный администратор](07-system-admin.md) |

## Справка

Интерфейс доступен на **русском** и **узбекском**. Переключение языка описано в

варианте: [02-resident.uz.md](02-resident.uz.md).
