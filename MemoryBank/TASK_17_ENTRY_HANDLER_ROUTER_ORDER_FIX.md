# ИСПРАВЛЕНИЕ: Порядок регистрации роутеров

**Дата**: 6 ноября 2025  
**Проблема**: Работает только кнопка "Ariza yaratish", русская кнопка не работает

---

## 🔍 ПРОБЛЕМА

Handler срабатывает только для узбекского текста "📝 Ariza yaratish", но не для русского "📝 Создать заявку".

Логи показывают:
- ✅ Handler сработал для "📝 Ariza yaratish"
- ❌ Handler НЕ сработал для "📝 Создать заявку"

---

## 🔧 ПРИЧИНА

Проблема в порядке регистрации роутеров в `main.py`:

**БЫЛО**:
1. `onboarding_router` (строка 237) - регистрируется раньше
2. `requests_router` (строка 245) - регистрируется позже

В `onboarding.py` есть handler `process_manual_phone`, который проверяет системные команды, включая "📝 Создать заявку". Хотя этот handler работает только в состоянии `OnboardingStates.waiting_for_phone`, порядок регистрации может влиять на обработку сообщений.

---

## ✅ РЕШЕНИЕ

Изменен порядок регистрации роутеров:

**СТАЛО**:
1. `requests_router` - регистрируется раньше (сразу после `auth_router`)
2. `onboarding_router` - регистрируется позже

Это обеспечивает, что handler создания заявки (`start_request_creation`) срабатывает до handlers онбординга.
