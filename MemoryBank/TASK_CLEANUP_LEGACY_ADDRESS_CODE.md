## 📋 КОНТЕКСТ

После миграции на новую систему адресов (Справочник: Двор → Дом → Квартира) пользователи больше **НЕ МОГУТ** вводить адреса вручную, только выбирают из готовых вариантов.

В кодовой базе осталось **~400+ строк** устаревшего кода:
- Функции валидации ручного ввода
- Обработчики FSM для ввода адресов
- Устаревшие FSM-состояния
- Хелпер-функции форматирования
- Константы legacy типов адресов

## 🔄 ROLLBACK PLAN

### В случае проблем после деплоя:

1. **Git revert:** Откатить коммит с удалением
2. **Проверить логи:** Найти cause of failure
3. **Fix forward:** Исправить проблему и повторить
4. **Hotfix:** Временно вернуть critical функции

### Критерии для rollback:
- Ошибки при создании заявок
- Ошибки при регистрации пользователей
- Критические ошибки в логах (> 10/минуту)
- Невозможность выбрать адрес
